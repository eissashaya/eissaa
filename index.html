من أنا؟

مرحبًا، أنا عيسى إبراهيم شايع

مبرمج تطبيقات موبايل ومواقع ويب، متخصص في بناء الأنظمة الحديثة وتطوير الحلول الرقمية بأعلى جودة. أمتلك خبرة قوية في تطوير تطبيقات الموبايل باستخدام Flutter & Dart، إلى جانب تصميم وبرمجة مواقع ويب احترافية باستخدام HTML، CSS، JavaScript، وPHP , Node.js, React .

أعمل على تطوير المشاريع من الصفر، ابتداءً من تحليل الفكرة وتصميم الواجهات، وصولًا إلى البرمجة والتسليم النهائي. هدفي هو تحويل الأفكار إلى منتجات رقمية ناجحة، مع التركيز على الأداء، الأمان، وسهولة الاستخدام. يمكنك متابعة أعمالي وزيارة موقعي الشخصي أو حساباتي على منصات التواصل الاجتماعي.

Flutter Dart Python HTML / CSS JavaScript PHP React Node.js Bootstrap تطوير الويب تطبيقات الموبايل

تطبيق تحدي الويب

تطبيق تفاعلي ممتع لاختبار وتحسين مهاراتك في تطوير الويب! يتضمن سلسلة من التحديات البرمجية التي تساعدك على صقل مهاراتك في HTML، CSS، JavaScript و PHP.

سواء كنت مبتدئاً أو مطوراً محترفاً، هذا التطبيق سيساعدك على تحسين مهاراتك من خلال تحديات عملية وتمارين تفاعلية تغطي جميع جوانب تطوير الويب.
